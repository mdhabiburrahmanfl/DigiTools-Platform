# DigiTools

DigiTools is a responsive digital tools buying website built for exploring premium digital products, adding them to a cart, and completing a simple checkout flow. The interface is inspired by a modern marketplace layout and is designed to work smoothly across desktop, tablet, and mobile devices.

## Technologies Used

- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify

## Key Features

- Browse six premium digital products in a responsive three-column layout with pricing, tags, features, and icons.
- Add products to the cart, view the live cart count in the navbar, remove selected items, and clear the cart through checkout.
- Receive toast notifications for add-to-cart, remove, and checkout actions while navigating a polished landing page with stats, steps, pricing, and footer sections.
